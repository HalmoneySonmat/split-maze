# SPLIT-MAZE

> procgen 미로 RL 에이전트 + from-scratch 합성 미로언어 LM을 인공 뇌량(ACC)으로
> 결합해, 에이전트의 실제 내부 목표(목표 오일반화 상황 포함)를 LM이 합리화가
> 아니라 충실하게 말할 수 있는지 검증하는 프로젝트.

**상태**: Phase 0 (설계 완결, 환경 구축 중). 설계 전문은 [`PLAN.md`](PLAN.md) 참조.

(이질적인 진짜 환경, 양쪽 다 from scratch).
